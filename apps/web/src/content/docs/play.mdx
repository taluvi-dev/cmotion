A live editor for cmotion. The source on the left is parsed, type-checked,
and evaluated by `cmotion-render.wasm` — the same interpreter `cmo` runs
on the CLI — and the JSON value tree it emits is translated into Three.js
for the preview on the right.

<PlayEmbed />

The video duration is read from the source itself
(`duration: Duration = 6s`), so **Save video** records exactly what
the `.cm` declares.
