A 16-frame character sheet (idle, walk, attack, death — one row each)
packed into a 4×4 grid, loaded from `/img/viking.png` and drawn with
`sprite(...)`.

`image(src)` is the source value — a path to an image, or an inline
`data:image/png;base64,…` URI for small sheets — and `sprite(image(src),
width:, height:, cols:, rows:, frame:)` is the displayable textured quad.
`cols`/`rows` describe the atlas grid; `frame` selects a cell (0 =
top-left, row-major) and is floored, so an animated `frame:` steps crisply
rather than cross-fading.

Here `frame:` ramps across all 16 cells on a looping `animate`, so the
viking runs through every action — idle, walk, attack, death. Because this
sheet has a solid white cell background and its frames aren't grid-aligned,
`key: #ffffff` cuts out the background and `anchor: center` re-centres each
frame's content so the figure stays put instead of wandering.
